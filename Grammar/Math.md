# Math 함수

올림 : Math.ceil(a)

- 매개변수 타입 : `double`
- 반환 타입 : `double`

반올림 : Math.round(a)

- 매개변수 타입 : `float`, `double`
- 반환 타입 : `float`, `double`

제곱근 : `Math.sqrt(a)` =

- 매개변수 타입 : `double`
- 반환 타입 : `double`

큰 값 : Math.max(a, b) = a와 b 중 큰 값

- 매개변수 타입 : `int`, `long`, `float`, `double`
- 반환 타입 : `int`, `long`, `float`, `double`

내림 : Math.floor(a)

- 매개변수 타입 : `double`
- 반환 타입 : `double`

제곱 : `Math.pow(a, b)` =

- 매개변수 타입 : `double`
- 반환 타입 : `double`

절대값 : Math.abs(a)

- 매개변수 타입 : `int`, `long`, `float`, `double`
- 반환 타입 : `int`, `long`, `float`, `double`

작은 값 : Math.min(a, b) = a와 b 중 작은 값

- 매개변수 타입 : `int`, `long`, `float`, `double`
- 반환 타입 : `int`, `long`, `float`, `double`
